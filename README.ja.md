# 地方公共団体へのオープンデータの取組に関するアンケート (opendata_survey)

本リポジトリには、日本の地方公共団体を対象に実施されたオープンデータの取組に関するアンケート結果が含まれています。政府CIOポータルによって元々公開されていたデータを、分析や閲覧が容易なCSVおよびMarkdown形式に加工しています。

また、集計されたアンケート結果を可視化するインタラクティブなWebアプリケーションも提供しています。

## データの閲覧

- **📊 [インタラクティブなアンケート結果ダッシュボード](https://code4fukui.github.io/opendata_survey/)**
  - 動的なグラフや表を用いて、集計されたアンケートデータを閲覧できます。

- **📄 [地方公共団体別の回答を閲覧](https://github.com/code4fukui/opendata_survey)**
